# IDD-Lite-Unet

# Training procedure 

1. Download the dataset in a folder "dataset" and arrange the data in the following structure:

```bash
├── dataset
│   ├── images
│   │   ├── train
│   │   ├── val
│   │   ├── test
│   ├── annotation
│   │   ├── train
│   │   ├── val
```
2. execute the command 
```bash
bash run.sh
```
